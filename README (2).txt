# 🖐️ AI Gesture Control System

## 📌 Project Overview

The **AI Gesture Control System** is a real-time computer vision project developed using **Python** that allows users to control system functions using hand gestures detected through a webcam.

This system enables intuitive and touchless interaction with the computer by recognizing hand gestures and mapping them to system actions such as volume control, brightness adjustment, screenshots, and cursor movement.

The project demonstrates practical implementation of **Computer Vision**, **Human-Computer Interaction (HCI)**, and **Automation** using modern Python libraries.

---

## 🚀 Features

* Real-time hand gesture detection using webcam
* Gesture-based Volume Control
* Gesture-based Brightness Control
* Screenshot capture using gesture
* Cursor movement using hand tracking
* Smooth and responsive user interface
* Reduced false triggers using gesture filtering
* Multithreading for better performance
* System monitoring dashboard
* User-friendly graphical interface

## 📦 Installation

### Step 1 — Install Python

Download and install Python:

https://www.python.org/downloads/

During installation:

✔ Check **Add Python to PATH**

---

### Step 2 — Clone the Repository

```bash
git clone https://github.com/your-username/AI-Gesture-Control-System.git
```

---

### Step 3 — Navigate to Project Folder

```bash
cd AI-Gesture-Control-System
```

---

### Step 4 — Install Required Libraries

```bash
pip install PyQt5 opencv-python mediapipe pyautogui psutil screen-brightness-control pycaw comtypes
```

---

## ▶️ How to Run the Project

Run the main Python file:

```bash
python main.py
```

After running:

* Webcam will start automatically
* Gesture detection will begin
* System controls will activate

## ✋ Gesture Controls

| Gesture                  | Action             |
| ------------------------ | ------------------ |
| Index finger move        | Cursor movement    |
| Thumb touch index finger | Click              |
| Fingers very close       | Screenshot         |
| Hand move up/down        | Volume control     |
| Hand move left/right     | Brightness control |
| Hand wide open           | Next tab           |

## 🛠️ Troubleshooting

### Webcam not opening

Check:

* Camera permissions enabled
* No other app using camera

---

### Module not found error

Install missing libraries:

```bash
pip install module-name
```

Example:

```bash
pip install mediapipe
```

---

### Low performance / Lag

Try:

* Closing background applications
* Using good lighting conditions
* Reducing camera resolution
